About This Blog

Real web development tutorials for people who want to learn by building.

Mission

Build Apps Blog exists to share practical, step-by-step tutorials on building real frontend applications. Every tutorial covers a complete project — from planning and architecture through to production-ready code.

We believe the best way to learn web development is by building things that actually work. No toy examples. No single-concept deep-dives without context. Just real projects with real code.

About Japhet Sunday

Hi! I'm Japhet Sunday, a frontend developer from Nigeria with an HND in Mass Communication. I discovered web development and fell in love with the craft of building things on the web.

I started this blog to share what I've learned building real projects — and to help others avoid the pitfalls I hit along the way. Every tutorial here is a project I've actually built, debugged, and deployed.

When I'm not coding, I'm writing, learning new frameworks, or thinking about the future of web development.

What You'll Find Here

  • JavaScript Projects — expense trackers, dashboards, chat apps, and more
  • CSS Techniques — layouts, animations, responsive design, and modern CSS features
  • UI Components — navbars, modals, forms, and interactive elements
  • Frontend Patterns — state management, API integration, localStorage strategies
  • Fundamentals — the core concepts that everything else builds on

About This Site

Build Apps Blog is built entirely with vanilla HTML, CSS, and JavaScript. No frameworks. No build tools. No dependencies. Just clean, semantic HTML and standards-based JavaScript.

This is intentional. We walk the walk here — if we're teaching vanilla JS, we use vanilla JS.
